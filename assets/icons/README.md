# Icons Directory

This directory contains icon files used by the Floor Monitoring Application.

## Icon Files

- `app_icon.ico` - Main application icon
- `camera_icon.png` - Camera-related icons
- `worker_icon.png` - Worker-related icons
- `alert_icon.png` - Alert-related icons
- `report_icon.png` - Report-related icons
- `settings_icon.png` - Settings-related icons

## Icon Requirements

Icons should be in the following formats:
- `.ico` for Windows application icons
- `.png` for UI elements (32x32 or 64x64 recommended)

## Adding Icons

To add custom icons:
1. Create your icon files in the appropriate format
2. Place them in this directory
3. Update the UI code in `src/ui_manager.py` to use the new icons
